# Power BI Secure Login Portal (Full Stack)

**Live Static Preview:** https://tirthmodi95.github.io/my-portfolio/

**Full Dynamic Version Features:**
- User registration & secure login
- Admin panel with real-time user management
- Activity logging
- Secure password reset
- Real MongoDB backend
- Responsive dark theme design

**Tech Stack:**
- Django 6.0 + Django REST Framework
- MongoDB (official backend)
- HTML/CSS/JavaScript + Chart.js

**To run locally:**
```bash
git clone https://github.com/Tirthmodi95/my-portfolio.git
cd my-portfolio/powerbi_project
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
